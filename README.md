# 💼 Salary Prediction Web App

A Machine Learning web application that predicts employee salary based on **Years of Experience** using **Linear Regression** and deploys the model through **Streamlit**.

---

## 📌 Project Overview

This project takes user input for years of experience and predicts the estimated salary instantly.

It helps understand:

- Data preprocessing
- Linear Regression
- Model training
- Model evaluation
- Streamlit deployment
- ML + Web App integration

---

## 🚀 Features

✅ Takes Years of Experience as input  
✅ Uses Linear Regression Model  
✅ Predicts Salary Instantly  
✅ Interactive Streamlit UI  
✅ Data Visualization  
✅ Model Performance Metrics  
✅ Regression Line Graph

## 📈 Machine Learning Model

This project uses:

**Linear Regression**

Formula used:

y = mx + c

Where:

- y = Salary
- x = Years of Experience
- m = slope
